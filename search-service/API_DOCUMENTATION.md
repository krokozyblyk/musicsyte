# Search Service API Documentation

**Base URL:** `http://localhost:8086` (или через API Gateway: `http://localhost:8080/api`)

**Service Port:** 8086

## Статус

⚠️ **API еще не реализовано**

Контроллер для Search Service находится в разработке. После реализации здесь будет полная документация по всем endpoints.

## Планируемый функционал

- Глобальный поиск по всем сущностям
- Поиск треков
- Поиск альбомов
- Поиск исполнителей
- Поиск плейлистов
- Расширенный поиск с фильтрами
- Автодополнение запросов

---

## Примечания

- API будет доступен как напрямую на порту 8086, так и через API Gateway на порту 8080
- Сервис будет обеспечивать быстрый полнотекстовый поиск по всей базе данных
