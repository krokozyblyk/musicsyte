# Analytics Service API Documentation

**Base URL:** `http://localhost:8081` (или через API Gateway: `http://localhost:8080/api`)

**Service Port:** 8081

## Статус

⚠️ **API еще не реализовано**

Контроллер для Analytics Service находится в разработке. После реализации здесь будет полная документация по всем endpoints.

## Планируемый функционал

- Запись истории прослушиваний
- Статистика пользователей
- Аналитика по трекам
- Аналитика по исполнителям
- Отчеты и метрики

---

## Примечания

- API будет доступен как напрямую на порту 8081, так и через API Gateway на порту 8080
- Сервис будет использовать базу данных для хранения аналитики
