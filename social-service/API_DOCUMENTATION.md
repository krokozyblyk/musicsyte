# Social Service API Documentation

**Base URL:** `http://localhost:8087` (или через API Gateway: `http://localhost:8080/api`)

**Service Port:** 8087

## Статус

⚠️ **API еще не реализовано**

Контроллер для Social Service находится в разработке. После реализации здесь будет полная документация по всем endpoints.

## Планируемый функционал

- Подписки на пользователей (follow/unfollow)
- Социальные взаимодействия (лайки, комментарии)
- Поделиться контентом
- Активность пользователей
- Социальные новости и лента

---

## Примечания

- API будет доступен как напрямую на порту 8087, так и через API Gateway на порту 8080
- Сервис будет использовать базу данных для хранения социальных связей и взаимодействий
