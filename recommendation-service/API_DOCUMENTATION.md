# Recommendation Service API Documentation

**Base URL:** `http://localhost:8085` (или через API Gateway: `http://localhost:8080/api`)

**Service Port:** 8085

## Статус

⚠️ **API еще не реализовано**

Контроллер для Recommendation Service находится в разработке. После реализации здесь будет полная документация по всем endpoints.

## Планируемый функционал

- Получение рекомендаций для пользователя
- Рекомендации на основе истории прослушиваний
- Рекомендации на основе похожих пользователей
- Популярные треки
- Персонализированные рекомендации

---

## Примечания

- API будет доступен как напрямую на порту 8085, так и через API Gateway на порту 8080
- Сервис будет использовать алгоритмы машинного обучения для генерации рекомендаций
